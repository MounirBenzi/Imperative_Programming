Description: This is a program that visualises different data types by converting them into binary

Usage: This program can be executed from the command line, using the line:  $ ./rpn.exe operands... operators

            For example:
            $ ./visualise char 8
            -8 binary digits: 00001000
            Ones Compliment: 11110111
            Twos Compliment: 11111000


The program can do this for char, long, int and short data types
